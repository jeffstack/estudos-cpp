# 📚 Exercícios de Strings - C++

Este diretório contém uma lista de exercícios focados em manipulação de strings em C++, abordando entrada de dados, formatação de texto, leitura de arquivos e processamento de strings.


## 🧪 Exercício 1 - Cascata de Mensagem

Receber através de digitação uma mensagem.

Apresentar esta mensagem centralizada na linha 5 da tela.

A partir daí, cada letra desta mensagem deverá "cair", da linha 5 até a linha 20, simulando uma "cascata".  
Ao final, a mensagem deverá estar completamente posicionada na linha 20.


## 🔎 Exercício 2 - Verificação de Palíndromo

Receber através de digitação uma mensagem.

Informar se esta mensagem é ou não um palíndromo.

Observação:
- Espaços devem ser ignorados na verificação.

Exemplos:
- OVO → é palíndromo  
- ARARA → é palíndromo  
- SOCORRAM ME SUBI NO ONIBUS EM MARROCOS → é palíndromo  


## 📄 Exercício 3 - Formato de Agenda Telefônica

A partir de uma lista de nomes disponível em um arquivo `.txt`, transformar os nomes para o formato de agenda telefônica.

Exemplo:

Entrada:
José Antonio Santos Silva

Saída:
Silva, José Antonio Santos


## 📚 Exercício 4 - Citação Bibliográfica

A partir de uma lista de nomes disponível em um arquivo `.txt`, transformar os nomes para o formato de citação bibliográfica.

Exemplo:

Entrada:
José Antonio Santos Silva

Saída:
SILVA, José A. S.


## 🗂️ Objetivo da Lista

Praticar:

- Manipulação de strings em C++
- Formatação de texto
- Tratamento de entrada e saída
- Leitura de arquivos `.txt`
- Transformação de dados textuais


## 📌 Observação

Os exercícios 3 e 4 utilizam o arquivo `nomes.txt` como entrada de dados.

Esse arquivo contém uma lista de nomes, utilizada para leitura e transformação dos dados nos formatos solicitados.

O arquivo `nomes.txt` deve estar no mesmo diretório dos executáveis, ou o caminho do arquivo deve ser ajustado no código.
