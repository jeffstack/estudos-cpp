# 🎟️ Sistema de Bilheteria
Mini-projeto com foco na prática de alocação dinâmica de memória e manipulação de matrizes por meio de ponteiro para ponteiro.

## 📌 Sobre
O programa simula um sistema de reserva de poltronas de um teatro, permitindo a visualização do mapa de ocupação, a reserva de assentos e a exibição de informações relacionadas ao faturamento.

## 🛠️ Funcionalidades
- Reserva de assentos
- Exibição do mapa de poltronas
- Verificação de ocupação
- Cálculo de faturamento
- Interação por menu no terminal

## 💡 Conceitos praticados
- Estruturas condicionais e de repetição
- Funções
- Matrizes
- Ponteiro para ponteiro
- Alocação dinâmica de memória
- Organização lógica do programa

## ▶️ Como executar
1. Compile o arquivo `.cpp`
2. Execute o programa no terminal
3. Escolha as opções disponíveis no menu
